# 📊 Supply Chain Management Dashboard (Power BI)

## 📌 Project Overview

This project presents an end-to-end Supply Chain Analytics dashboard built using Microsoft Power BI. It provides insights into supplier performance, cost efficiency, lead time trends, and product distribution.

The project is divided into multiple dashboards:

* **Dashboard 1:** Core KPIs and overview metrics
* **Dashboard 2:** Advanced insights, trends, and supplier comparison

---

## 🎯 Objectives

* Monitor supply chain performance
* Identify high-cost and high-risk suppliers
* Analyze lead time trends over time
* Track defect rates and quality issues
* Enable interactive filtering and comparison

---

## 🧰 Tools & Technologies

* Microsoft Power BI Desktop
* DAX (Data Analysis Expressions)
* Excel Dataset

---

## 📊 Dashboard 1 – Overview

### Key KPIs:

* Total Cost
* Daily Orders
* Monthly Orders
* Ordered Quantity
* Min/Max Cost per Unit

### Insights:

* Supplier-wise performance
* Defect rate by inspection results
* Revenue distribution
* Cost across locations

---

## 📈 Dashboard 2 – Advanced Insights

### 🔹 Lead Time Trend

* Line chart showing average lead time by month
* Helps identify delays and seasonal patterns

### 🔹 Defect Rate by Supplier

* Identifies high-risk suppliers
* Helps in quality control decisions

### 🔹 Cost vs Quantity Analysis

* Scatter plot:

  * High quantity + low cost → Best suppliers
  * High cost + low quantity → Risky suppliers

### 🔹 Supplier Ranking

* Based on:

  * Total Cost
  * Lead Time
  * Monthly Orders

### 🔹 Contribution Analysis

* % of total cost by supplier
* % of ordered quantity
* Helps identify dependency risks

### 🔹 Product Distribution

* Ordered quantity % by supplier and product type

---

## 📌 Key Business Insights

* Suppliers with high defect rates need review
* Lead time varies across months → possible seasonal delays
* Some suppliers dominate cost → risk of dependency
* Best suppliers = low cost + high volume + low defect rate
